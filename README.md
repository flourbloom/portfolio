# [Seavhong CHEA Portfolio](https://flourbloom.github.io/portfolio/)

This is a personal portfolio website for Seavhong CHEA, showcasing skills in software engineering, UI/UX design, and featured projects.

## Structure

- `index.html` — Home page (About, Services, Education, Achievements, Testimonials)
- `projects.html` — Featured Projects
- `contact.html` — Contact information and form
- `assets/` — Contains CSS, JS, and images

## How to Use

1. Open any HTML file in your browser (recommended: start with `index.html`).
2. Use the navigation bar to switch between Home, Projects, and Contact pages.
3. The contact form allows visitors to send a message (demo only; no backend integration).

## Technologies

- HTML5
- CSS3
- JavaScript
- [Font Awesome](https://fontawesome.com/)
- [AOS Animation](https://michalsnik.github.io/aos/)
